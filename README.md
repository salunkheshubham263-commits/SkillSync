# SkillSync

SkillSync is a full-stack student networking platform inspired by LinkedIn, designed to help students showcase their skills, share projects, and connect with like-minded peers.

> ⚠️ Project Status: Currently Under Progress

---

## 🌟 Features

- User Authentication (Login / Signup)
- User Profiles
- Post Creation & Feed System
- Like & Comment Functionality
- Follow / Unfollow Users
- Skill-Based Networking
- Responsive UI
- Smooth Animations using GSAP

---

## 🛠️ Tech Stack

### Frontend
- React.js
- GSAP
- CSS

### Backend
- Node.js
- Express.js

### Database
- PostgreSQL

### DevOps
- Docker (Planned)

---

## 🎨 Design Philosophy

SkillSync focuses more on:
- Usability
- Clean User Experience
- Minimal Modern UI
- Smooth Interactions

instead of overly complex designs.
